Introduction

This is a classic memory game where the player have to find all the pairs on the board. The player have to click on every cards, if the cards are matchig the cards stay flipped, if they are not pairs they will turn back. The aim of the game is to find all the pairs of the same images with the less moves what possible. The game is over when all cards flipped over. To restart the game use reset button.
Enjoy!
